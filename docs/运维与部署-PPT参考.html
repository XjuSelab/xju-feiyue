部署 · 生产架构

讲者 E
一台华为云 VPS，nginx 静态服 + 反代 FastAPI，systemd 托管，脚本一键发。
服务器与拓扑华为云 VPS（huawei2）；nginx 同机做静态服 frontend/dist + 反代 FastAPI；链路 DNS → nginx →〔静态 / 反代〕→ uvicorn → SQLite。
多域名winbeau.top（主）+ feiyue.selab.top（镜像），同一 VPS 多域名共存。
进程托管systemd feiyue-backend.service 管 uvicorn（:8001），开机自启、崩溃自动拉起。
一键部署deploy.sh：git pull main → uv sync → alembic upgrade head → systemctl restart → /health 探活 → pnpm build。
1 台 VPS2 域名uvicorn :8001deploy.sh 6 步

可选：贴一段真实配置更有说服力（示例，以生产实际为准）

# nginx：静态直出 + 反代（示意）
server {
  server_name winbeau.top feiyue.selab.top;
  root /srv/xju-feiyue/frontend/dist;
  location / { try_files $uri /index.html; }          # SPA 前端
  location /api/ { proxy_pass http://127.0.0.1:8001/; } # 反代后端
  location /uploads/ { proxy_pass http://127.0.0.1:8001/uploads/; }
}
# systemd：feiyue-backend.service（示意）
[Service]
ExecStart=uv run uvicorn app.main:app --host 127.0.0.1 --port 8001
Restart=always

我负责部署与运维。整个平台跑在一台华为云 VPS 上，nginx 同时做两件事：把前端构建产物 frontend/dist 作静态资源直接发出去，同时把 /api 反向代理到后端。后端是 FastAPI，用 uvicorn 跑在 8001 端口，由 systemd 的 feiyue-backend 服务托管，开机自启、崩溃自动拉起。上线只要一条 deploy.sh：拉主分支、uv 同步依赖、alembic 迁移、重启服务、健康探活、再构建前端。winbeau.top 是主域名，feiyue.selab.top 是镜像，都指到这台机器。

运维里最要命的是数据不能丢。我把数据库、上传附件和密钥全部同步到一个私有的 HuggingFace 数据集 winbeau/xju-feiyue-data，异地留存。里面按命名空间分区：state 放实时状态，schools 和 conf 放高校导师库和会议库，各自独立。日常备份由 make sync-push 定时 cron 完成；一旦换机或需要灾备，一条 make data-pull 就能把整站数据拉回来复现。内容真源是 62 篇 seed markdown，加上站内用户直接发布的笔记。
S16

运维 · 安全与性能

讲者 E
鉴权 / 文件安全 / 性能预算 / 管理后台四道防线，保证平台稳与快。
鉴权python-jose HS256 JWT（7 天有效）+ bcrypt 密码哈希。
文件安全（双重防存储型 XSS）上传拦截 .svg/.html/.htm/.xml（DENY_EXTS）+ HardenedStaticFiles 给 /uploads 每个响应注入 X-Content-Type-Options: nosniff；再加 50MB 上限 + magic-byte 嗅探。
性能首屏 JS gzip ≈ 144KB、全路由 React.lazy + Suspense 懒加载、AI 欢迎语 DeepSeek 缓存 3h 轮换省 API。
管理后台登录事件审计、数据热 reload、会议爬取手动触发。
JWT 7 天首屏 ≈144KB缓存 3h上传 ≤50MB
占位符（你自己截）：①管理后台「登录事件审计 / 数据热 reload / 会议手动触发」界面截图；②可选 pnpm build 的 bundle 体积输出截图，佐证 144KB。我无法访问运行中的后台，只能替你画预算柱状图。
安全和性能上我做了四道防线。鉴权用 HS256 的 JWT，7 天有效，密码走 bcrypt 哈希。文件安全上，上传时拦掉 svg、html 这类能被当脚本执行的扩展名，静态服务再对每个响应统一注入 nosniff 头，双重防存储型 XSS，另外还有 50MB 上限和 magic-byte 校验。性能上首屏 JS 压缩后约 144KB，全路由懒加载，AI 欢迎语缓存 3 小时轮换、省下大量 API 调用。管理后台能审计登录事件、热更新数据、手动触发会议爬取。
