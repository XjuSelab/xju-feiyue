## 合并到 Word 的注意事项

- 图源文件统一使用 drawio：在 app.diagrams.net 打开 `.drawio` 后导出 PNG/SVG，再插入 Word；需要修改图时直接改 `.drawio` 源文件。
- 图题在正文 md 中（"图 4-x ……"行），插入 Word 后按课程格式要求置于图下方、五号宋体加粗居中。
- 正文中「〔待填〕」为成员姓名/学号占位，合并时替换。
