# 飞跃 · Feiyue

> 新疆大学 SeLab 的笔记站。学长姐写长文，把踩过的坑、走通的路留给下一届。

线上：<https://winbeau.top>

---

## 里面有什么

七个分类，每个分类下都是按标签 / 时间检索的长文笔记。

| 分类   | 大概会看到 |
|---|---|
| **科研** | 论文精读、方向选择、第一次跑通 baseline |
| **课程** | 离散 / 编译 / 408 / 数据库实验，上一届的复习笔记 |
| **推荐** | 读过的书 / 论文 / 教程，附一句为什么值得读 |
| **竞赛** | 数模 / ICPC / 挑战杯，从报名到答辩 |
| **Kaggle** | 入门 / notebook 组织 / 上分思路 |
| **工具**   | 服务器 / Docker / Git / VS Code Remote |
| **生活**   | 食堂 / 选导师 / 出国申请 / 心理健康 |

笔记支持 Markdown、KaTeX 公式、围栏代码块；个人页能看到这个人写过的所有笔记。

## 想加一篇笔记

笔记真源是 `content/notes/*.md`——一篇笔记就是一个 markdown 文件。
两种方式：

- **fork + PR**：往 `content/notes/` 扔 `.md` 文件，frontmatter 字段见 [`CONTRIBUTING.md`](CONTRIBUTING.md)
- **在站内写**：联系站长 [@winbeau](https://github.com/winbeau) 开账号，直接在网站发布
