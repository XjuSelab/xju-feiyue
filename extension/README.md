# 飞跃 · 成绩单一键导入（浏览器扩展）

在新疆大学教务系统成绩页加一个**悬浮按钮**，点一下就把成绩单导出并回传到飞跃「学分统计」，自动出结果。不用书签、不用控制台。

## 安装（Edge / Chrome，一次即可）

1. 把本 `extension/` 文件夹整个放到你电脑上（任意位置）。
2. 地址栏打开扩展页：Edge 输 `edge://extensions`，Chrome 输 `chrome://extensions`。
3. 打开右上角/左下角的 **「开发者模式」**。
4. 点 **「加载解压缩的扩展」**（Load unpacked），选中这个 `extension/` 文件夹。
5. 安装完成，列表里会出现「飞跃 · 成绩单一键导入」。

## 使用

1. 浏览器里**登录教务系统**，打开任意教务页面（推荐「查看成绩」页）。
2. 右下角出现绿色悬浮按钮 **「📥 导入飞跃」**，点它。
3. 按钮变「导出中…」→「✅ 已回传，正在打开飞跃…」，并自动打开飞跃学分统计页。
4. 飞跃页**一进去就自动取回并解析**，出各模块学分与达标检查。
   - 若飞跃页要求登录，登录后会自动取回（后端暂存 5 分钟）。

## 原理与安全

- 扩展只在 `jwxt-443.webvpn.xju.edu.cn` 页面运行，用的是**你当前已登录的教务会话**，全程不接触你的密码。
- 导出的 PDF 经飞跃后端 `/notes/transcript-stash` **按学号暂存 5 分钟**、仅本人可取回、取后即删。
- 仅声明了对教务系统域名的 content script，无其它权限。

> 学号、入学年级（`rxnj` 取学号前 4 位）自动适配各年级；导出的是「已选已修成绩 · 入学以来」。
